== Installation ==

**Custom one Click SEO Sitemap [Installation Guide]**

1. You can:
 * Upload the 'custom-one-click-seo-sitemap' folder to '/wp-content/plugins/' directory via FTP. 
 * Upload the full ZIP file via *Plugins -> Add New -> Upload* on your WordPress Administration Panel.
 * Search **Custom one Click SEO Sitemap** in the search engine available on *Plugins -> Add New* and press *Install Now* button.
2. Activate plugin through *Plugins* menu on WordPress Administration Panel.
3. Ready, now you can enjoy it, and if you like it and find it useful.
